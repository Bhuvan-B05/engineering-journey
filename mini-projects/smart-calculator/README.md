# Smart Calculator

A command-line calculator built using Python with support for arithmetic operations, history tracking, file persistence, and exception handling.

## Features

- Addition, subtraction, multiplication, division
- Floor division, modulus, and power operations
- History tracking using lists
- Save/load history using `history.txt`
- Exception handling for invalid input
- Division-by-zero protection
- Continuous command-line interaction
- Note:
  Trigonometric functions (`sin`, `cos`, `tan`) use radians.
